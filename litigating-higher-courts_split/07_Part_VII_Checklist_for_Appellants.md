## Part VII — Checklist for Appellants

### Pre-Filing
- [ ] Identify all issues for appeal
- [ ] Confirm all issues were preserved below
- [ ] Identify applicable standard of review for each issue
- [ ] Calculate and calendar all deadlines
- [ ] Order transcript (do this immediately)
- [ ] Research all controlling and persuasive authority

### Briefing
- [ ] Craft compelling questions presented
- [ ] Write accurate, record-supported statement of facts
- [ ] Write argument section with clear rule statements and applications
- [ ] Address adverse authority
- [ ] Stay within page/word limits
- [ ] Comply with all formatting requirements
- [ ] Include certificate of service and compliance

### Oral Argument Preparation
- [ ] Know the record cold — every fact, every page
- [ ] Prepare for hard questions — the most uncomfortable questions are the ones you must answer
- [ ] Practice answering questions in the middle of your argument (judges will interrupt)
- [ ] Prepare one-sentence answers to key record questions
- [ ] Know your strongest three points; be able to deliver them in any order
- [ ] Know the relief you want — state it clearly if asked
